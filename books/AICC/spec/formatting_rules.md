# AICC Amazon KDP Formatting Specification

Based on Amazon KDP Paperback Submission Guidelines (Retrieved: 2026-02-24).

## 1. File Requirements

- **Format:** PDF (preferred for consistent layout).
- **Fonts:** All fonts must be embedded in the PDF.
- **Images:** Minimum 300 DPI.
- **Flattening:** All transparency and layers must be flattened.
- **Cleanliness:** No crop marks, bookmarks, or placeholder text.

## 2. Page Setup & Margins

- **Trim Size:** Recommended 6" x 9" (Standard for business/strategic books).
- **Margins (based on 24-150 pages):**
  - **Inside (Gutter):** 0.375" (9.6 mm)
  - **Outside:** 0.25" (6.4 mm)
- **Bleed:**
  - If using images to the edge: Add 0.125" (3.2 mm) to top, bottom, and outside edges.
  - Manuscript PDF size for 6x9 with bleed: 6.125" x 9.25".

## 3. Interior Elements

- **Pagination:** Odd numbers on the right, even on the left.
- **Front Matter:**
  - Title Page
  - Copyright Page (LLC 2025, Abimbola Olaitan)
  - Dedication
  - Table of Contents
- **Typography:**
  - Minimum font size: 7pt (Body text recommended 10pt-12pt).
  - Line width: Minimum 0.75 point for charts/tables.

## 4. Cover Specifications (Pending Final Page Count)

- **Calculation:** Spine width depends on final page count and paper type (White/Cream).
- **Format:** Single PDF including Back + Spine + Front.
- **Bleed:** 0.125" on all sides.
