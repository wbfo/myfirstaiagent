# Surplus Funds Orchestrator (v2026 Autonomous)

This skill orchestrates the **Autonomous Recovery Engine**, transforming Surplus Funds from a manual workflow into an agent-led business.

## Agent Workflows

### 1. Researcher: Lead Acquisition

- Monitor foreclosure bulletins.
- Extract data from PDFs using Vision AI.
- Reference: `memory/recovery_engine/autonomous_recovery_engine.md`.

### 2. Market-Advisory: Intelligence

- Execute skip-tracing and LLC unmasking.
- Cross-reference social data (LinkedIn/Obituaries).

### 3. Deal-Closer: Outreach

- Craft one-to-one personalized hooks.
- Interface with Voice AI (Bland.ai/Retell) for verification.

### 4. Architect: Processing

- Map claim forms to PDF templates.
- Organize evidence packets.

### 5. Ops-Coordinator: Persistence

- Manage the 8-16 week claim tracking.
- Automated treasury follow-ups.

## Operational Standards

- **Asymmetric Outreach:** Always lead with the specific fund amount and case ID.
- **Capital Preservation:** No upfront fees; success-based model only.
