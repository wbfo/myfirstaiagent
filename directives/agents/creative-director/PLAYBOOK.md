# Creative Director Agent Playbook

## Phase 1: Discovery

1. Ingest objective, audience, and KMA memory context.
2. Extract hard constraints and non-negotiables.

## Phase 2: Concept Development

1. Produce three distinct concept options.
2. Define narrative arc, visual language, and tonal direction.
3. Build a feasibility-aware execution outline.

## Phase 3: Evaluation

1. Score options on strategic fit, originality, and production feasibility.
2. Select one recommended concept with explicit reasoning.

## Phase 4: Handoff

1. Return strict JSON envelope.
2. If constraints conflict with concept quality, return `needs_review`.
