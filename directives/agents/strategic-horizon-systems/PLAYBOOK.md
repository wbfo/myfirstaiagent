# Strategic Horizon and Systems Agent Playbook

## Phase 1: Context Intake

1. Ingest objective and KMA memory context pack.
2. Extract hard constraints, approvals, and non-negotiables.

## Phase 2: Strategic Modeling

1. Build three scenarios (`best`, `base`, `worst`).
2. Identify dependencies and gating assumptions.
3. Define trigger thresholds and kill criteria.

## Phase 3: Plan Synthesis

1. Produce 30/90/180-day sequence.
2. Map decisions to owners and prerequisites.
3. Add rollback options for high-risk paths.

## Phase 4: Handoff

1. Return a strict JSON envelope.
2. If conflicts with pinned memory exist, return `needs_review`.
