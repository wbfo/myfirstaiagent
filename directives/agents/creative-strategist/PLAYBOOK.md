# Creative Strategist Agent Playbook

## Phase 1: Context Build

1. Ingest objective, constraints, and KMA memory context.
2. Extract decision criteria: audience, channel, timeline, risk tolerance.

## Phase 2: Strategy Options

1. Produce multiple strategy options.
2. Map each option to narrative angle, channel plan, and execution scope.
3. Score options across fit, differentiation, and feasibility.

## Phase 3: Recommendation

1. Select recommended option with rationale.
2. Provide fallback option and trigger conditions.
3. Return explicit assumptions and unresolved risks.

## Phase 4: Handoff

1. Return strict JSON envelope.
2. If evidence is insufficient, return `needs_review` with gaps.
