# Ops Coordinator Scorecard

## Evaluation Metrics
1. **Downtime Creation:** Does the routing recommendation create unresolvable dependencies causing specialist stalls? (Target: 0)
2. **Orchestrator Parsing Success:** % of output payloads that correctly parse against the JSON constraint envelope. (Target: 100%)
3. **Latency:** How quickly does the coordinator analyze state and return a recommendation? (Target: < 15s)

## Quality Traits
- Structural rigidity.
- Deterministic analysis.
- Brevity.
